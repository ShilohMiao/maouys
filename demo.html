古墓丽影 / Tomb Raider

  • 平台 : PC / Mac / PS4 / Xbox One / PS3 / Xbox 360 / Linux
  • 开发商 : Crystal Dynamics, Inc. / Eidos-Montreal
  • 发行日期 : 2013-03-05
概述 : 由晶体动力开发、Square Enix发行的角色扮演风格动作冒险游戏，为古墓丽影系列的第十部作品。本作被视为该系列的一次重启，游戏中特别着墨于影响流行文化甚钜的主角劳拉·克罗夫特的个性和自我。《古墓丽影》最初于2013年3月5日在PlayStation 3、Xbox 360和Microsoft Windows等平台上发行、接着于2014年1月23日发行OS X版本、又于2016年4月27日发行Linux版本。

古墓丽影：崛起 / Rise of the Tomb Raider

  • 平台 : PC / PS4 / Xbox One / Xbox 360
  • 开发商 : Square Visual Works / Crystal Dynamics, Inc.
  • 发行日期 : 2015-11-10（Xbox One）
概述 : 本作由晶体动力开发，微软工作室和史克威尔艾尼克斯发行。是2013年发行的《古墓丽影9》的续作，故事承接前者，操作方式及故事流程也一脉相承，并有所提升。游戏包含中文版及中文配音。

美国众神 / American Gods

  • 出版社: 北京联合出版公司
  • 出版年: 2017-4
概述 : 你崇拜什么，你就是什么。
美国这片 土地上，新神与旧神的纷争不断升级。为了夺回人们的信仰，北欧主神奥丁穿越整个美国去联合旧神，与新神决一死战。一场众神之战即将爆发。

蜘蛛侠：英雄归来 / Spider-Man: Homecoming

  • 导演 : 乔·沃茨
  • 主演 : 汤姆·霍兰德 / 小罗伯特·唐尼 / 玛丽莎·托梅 / 迈克尔·基顿 / 雅各布·巴特朗
  • 上映日期 : 2017-09-08(中国大陆) / 2017-07-05(台湾) / 2017-07-07(美国)
影片讲述了蜘蛛侠在钢铁侠“导师”的帮助下对抗终极反派秃鹰的故事。经历了美队3中的复联内战后，彼得·帕克（汤姆·霍兰德 Tom Holland 饰）回到普通高中生活，并受到钢铁侠托尼·斯塔克（小罗伯特·唐尼 Robert Downey Jr. 饰）的指导与暗中关注。
